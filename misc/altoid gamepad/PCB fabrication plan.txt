# Altoids Gamepad - PCB Fabrication Package

## Overview
This package contains the complete PCB fabrication files for the Altoids Tin Retro Gamepad project, consisting of two boards:
- **Main Controller Board**: 90mm × 55mm, houses Pi Zero and all control electronics
- **Display Interface Board**: 60mm × 40mm, mounts 2.8" TFT display in top lid

## Manufacturing Notes

### Critical Specifications
- **Via Aspect Ratio**: Maximum 8:1 (1.6mm board thickness ÷ 0.2mm drill)
- **Annular Ring**: Minimum 0.1mm (0.45mm pad - 0.2mm drill ÷ 2)
- **Solder Mask Registration**: ±0.1mm
- **Silkscreen Registration**: ±0.15mm
- **Board Thickness Tolerance**: ±0.1mm critical for connector fit

### Quality Requirements
- **Electrical Test**: 100% net continuity testing
- **Visual Inspection**: IPC-A-600 Class 2 standard
- **Impedance Control**: Not required (low-speed digital)
- **Controlled Dielectric**: Standard FR4 acceptable

## Pre-Flight Checklist

Before submitting to manufacturer, verify:

### Design Rule Check (DRC)
- [ ] Minimum trace width: 0.2mm
- [ ] Minimum spacing: 0.2mm  
- [ ] Via size: 0.2mm drill minimum
- [ ] Annular ring: 0.1mm minimum
- [ ] Board outline: Closed polygon
- [ ] No overlapping components

### Electrical Rule Check (ERC)
- [ ] All nets connected
- [ ] No floating pins
- [ ] Power/ground connections verified
- [ ] Connector pinouts correct
- [ ] Polarity markings clear

### Manufacturing Compatibility
- [ ] Drill sizes available at manufacturer
- [ ] Layer stackup matches standard offering
- [ ] Surface finish compatible with assembly
- [ ] Fiducials present for assembly
- [ ] Test points accessible
